# 3D Print Files

STL files for the enclosure, panel, brackets, knobs, and mounts.

## First Draft Warning

The current 3D model is a first draft. The original build required cutting and sawing parts of the print to make everything fit. Expect to trim, sand, drill, or adjust the printed parts before final assembly.

Included files:

- `AV_joystick.zip` - contains `AV_joystick.stl`; zipped because the raw STL is too large for GitHub browser upload
- `base (2).stl`
- `bottom.stl`
- `F1_knobs.stl`
- `Faceplate Riser.stl`
- `faceplate.stl`
