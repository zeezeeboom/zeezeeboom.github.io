Kerbal Simpit Controller

A build guide for a custom Arduino Mega flight panel for Kerbal Space Program, with staging, telemetry, joystick attitude control, throttle encoder, action groups, and status LEDs.

Download Arduino Sketch Jump To Pin Map

What It Does

Joystick

Normal mode controls pitch and yaw. Clicking the joystick toggles roll mode, where the X axis controls roll only.

Stage System

The stage button starts a launch countdown before first stage, then performs instant staging. The LCD encoder click resets the 1-digit display to blank.

Telemetry LCD

The 16x2 I2C LCD shows ascent, orbit, maneuver, and target pages from KerbalSimpit telemetry.

Altitude Display

The 4-digit display shows USSR in standby, wait before data arrives, and altitude once connected.

Parts List

  • Arduino Mega 2560 or compatible board
  • 16x2 LCD with I2C backpack
  • 1x common-cathode 4-digit 7-segment display
  • 1x common-cathode 1-digit 7-segment display
  • Analog joystick module with VRx, VRy, SW, 5V, and GND
  • 2x EC11 rotary encoders with push switches
  • Momentary stage button
  • Toggle switches or buttons for SAS, RCS, brakes, lights, gear, action groups, and display mode
  • Status LEDs for SAS, RCS, orbit, and system-off indicator
  • Current-limiting resistors for LED and 7-segment segment lines
  • Breadboard, jumper wires, panel hardware, and USB cable
Use resistors on LEDs and 7-segment segments. Do not connect bare LEDs or display segments straight to Arduino pins without current limiting.

Required Software

  • Arduino IDE
  • Kerbal Space Program with the KerbalSimpit plugin installed
  • Arduino libraries: KerbalSimpit, LiquidCrystal_I2C, Encoder, and SevSeg

In Arduino IDE, install libraries from Sketch > Include Library > Manage Libraries. KerbalSimpit may need to be installed from its project files if it is not available in Library Manager.

Wiring Rules

  • All buttons and toggles use INPUT_PULLUP. Wire one side to the Arduino pin and the other side to GND.
  • All modules must share a common GND with the Arduino.
  • The LCD I2C backpack uses the Mega I2C pins: SDA on 20 and SCL on 21.
  • The joystick analog pins connect to A0 and A1. The joystick switch connects to pin 46 and GND.
  • The direct-wired 4-digit display is multiplexed by the Arduino, so minor flicker is possible under heavy serial or LCD activity.

Upload Steps

  1. Download sketch_apr18a.ino.
  2. Open it in Arduino IDE.
  3. Select Tools > Board > Arduino Mega or Mega 2560.
  4. Select the correct COM port under Tools > Port.
  5. Install the required libraries if Arduino IDE reports missing includes.
  6. Click Upload.
  7. Start KSP with KerbalSimpit installed and connect the serial device.

Troubleshooting

LCD does not display text

Check 5V, GND, SDA pin 20, SCL pin 21, and the I2C address. The code uses 0x27; some backpacks use 0x3F.

4-digit display is mirrored

The code uses digit order {30, 31, 32, 33} so USSR reads correctly. If yours is reversed, change the digit order back to {33, 32, 31, 30}.

4-digit display flashes or looks unstable

The display is directly multiplexed by the Arduino. The sketch refreshes it aggressively, but a driver module such as MAX7219, TM1637, or HT16K33 is the best way to make it rock solid.

Joystick click does not toggle roll mode

Make sure the joystick switch pin is wired to Arduino pin 46 and GND. The switch should read LOW when pressed.

KSP does not respond

Confirm KerbalSimpit is installed in KSP, the correct serial port is selected, and the Arduino sketch has uploaded successfully to an Arduino Mega 2560.
